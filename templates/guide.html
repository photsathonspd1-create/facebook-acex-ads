📖 Acex Ads Guide

คู่มือการใช้งาน Acex Ads — ระบบจัดการ Facebook Ads อัจฉริยะ

🚀 เริ่มต้นใช้งาน

1

สมัครสมาชิก / เข้าสู่ระบบ

ไปที่หน้า Settings แล้วสมัครสมาชิกด้วยอีเมล หรือเชื่อมต่อ Facebook OAuth

2

เชื่อมต่อ Facebook Token

ไปที่ Settings → Facebook Token แล้ววาง Access Token จาก Facebook Marketing API

3

เลือก Ad Account

เลือก Ad Account ที่ต้องการจัดการจาก dropdown ด้านบน

📊 ฟีเจอร์หลัก

📈 Dashboard

ดูภาพรวม performance ของ campaigns ทั้งหมด — impressions, clicks, spend, ROAS

⚡ Smart Scaling

ระบบวิเคราะห์และแนะนำการเพิ่ม/ลดงบประมาณอัตโนมัติตาม ROAS และ CTR

🎨 Fatigue Detection

ตรวจจับโฆษณาที่ creative หมดสภาพ (CTR ลด > 30%) พร้อมแนะนำให้ refresh

🤖 AdsGPT

AI Chatbot ช่วยวิเคราะห์ campaigns, เขียน ad copy, แนะนำกลยุทธ์ (ต้องมี OpenAI API key)

⚙️ Auto-Rules

ตั้งกฎอัตโนมัติ — เช่น "ปิด ad ถ้า CPC > ฿20" หรือ "เพิ่มงบ 20% ถ้า ROAS > 3x"
