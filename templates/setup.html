⚙️ Acex Ads Setup

ตั้งค่าระบบก่อนเริ่มใช้งาน — กรอกข้อมูลแล้วกดบันทึก

1 Facebook OAuth

ตั้งค่า Facebook App เพื่อเชื่อมต่อบัญชีโฆษณา

จาก developers.facebook.com → App Settings → Basic
จาก App Settings → Basic → Show App Secret
ต้องตรงกับที่ตั้งค่าใน Facebook App → Facebook Login → Settings

2 OpenAI API (AdsGPT)

ตั้งค่า OpenAI เพื่อใช้ AI Chatbot วิเคราะห์โฆษณา

จาก platform.openai.com → API Keys
